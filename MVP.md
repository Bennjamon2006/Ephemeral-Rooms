# Ephemeral Rooms

Una web donde los usuarios pueden crear salas de chat temporales que desaparecen cuando los usuarios se desconectan. Con acceso a través de un enlace directo o código de sala. Ideal para conversaciones rápidas y privadas sin necesidad de registro.

# MVP

- Crear una sala de chat temporal con un enlace único.
- Permitir a los usuarios unirse a la sala mediante el enlace.
- Permitir a los usuarios unirse a la sala mediante el código de la sala.
- Permitir a los usuarios enviar mensajes en la sala de chat.
- Eliminar la sala de chat después de un minuto de inactividad (sin mensajes ni usuarios conectados).
